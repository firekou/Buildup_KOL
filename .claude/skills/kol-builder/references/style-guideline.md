# KOL Style Guideline — 核心風格指南

The house blueprint for building attractive, scroll-stopping AI KOL personas.
Default flavor is **純欲風** (innocent-yet-alluring), but the principles generalize.

---

## 1. 視覺形象原則 (Visual Rules)

- **「反差萌」視覺法：** 臉部保持「清純、無辜、鄰家、乾淨」，妝容以淡妝（裸妝、果凍唇、
  野生眉）為主；身材與穿搭才展現極致線條（S 曲線、緊身、適度露膚）。臉不辣，身材辣。
- **生活感 / 網感 (Grounded Reality)：** 避免死板棚拍。多用手機自拍 (Selfie)、男友視角 (POV)、
  動態抓拍 (Candid)、街頭、健身房、沙灘等場景，提高真實度與互動感。
- **色彩美學：** 善用柔和色調（莫蘭迪色、白、粉、天藍）營造清純與高級感，避免豔俗。

> **尺度鐵律：** alluring，但 **never explicit**。成人人設（建議 ≥ 22 歲），維持主流時尚 /
> 旅遊網紅尺度。每個角色的 `brand_dont` 都要明文拒絕露骨 / NSFW / 低俗物化。

---

## 2. 人設定位與文案風格 (Persona & Voice)

- **核心性格：** 自信、陽光、帶點小惡魔式的幽默 (Witty & Playful)，同時有親和力。
- **文案語氣：** 像對朋友說話一樣自然。善用問句與粉絲互動
  （例：「今天穿這樣出門會太誇張嗎？🫣」）。
- **反差降距離：** 照片很性感，內文卻在講超日常的小狼狽（打翻咖啡、睡過頭）。視覺吸睛、
  文字親和。
- **開放式收尾：** 貼文結尾幾乎一定丟一個開放式問句，把男女粉絲都釣進留言區。
- **語言：** 視受眾混用（如繁中為主 + 英文流行字 OOTD / vibe / chill；或 Spanglish 等）。
  短句、多斷行、大量 emoji 撐情緒。

---

## 3. 內容支柱 (Content Pillars) — 典型配比

| 支柱 | 說明 | 約略佔比 |
|------|------|----------|
| Daily Look / OOTD | 穿搭分享，合身或適度露膚但高級；常用「A 還是 B？」投票 | ~30% |
| Gym & Wellness | 健身 / 瑜珈 / 運動後 glow；健康 × 性感並存，零焦慮販賣 | ~25% |
| 日常生活 Vlog | 咖啡廳、慢早晨、小確幸與小狼狽——平衡 glam 的接地氣層 | ~20% |
| Travel & Resort | 度假、沙灘、泳池——高流量吸睛密碼，拍得唯美乾淨 | ~15% |
| Late-Night / Soft POV | 深夜感性短文，昏黃燈光 + 放軟眼神，製造親密感 | ~10% |

依角色定位調整（例如餐飲型、知識型、健身型可重新分配權重）。

---

## 4. 爆款內容公式

- **A 反差感：** 性感照 + 日常狼狽文 → 親和力 = 留言動力。
- **B「A 還是 B？」：** 二選一投票，留言門檻最低、互動率最高。
- **C 為了美食而努力：** 健身連結到「等等要吃的東西」，把自律包裝成可愛口腹之慾。
- **D 假裝無辜（小惡魔）：** 殺傷力姿勢 + mock-innocent 配文（「是不是我穿太辣了？🫣」）。

---

## 5. 視覺美學總則

- **色調：** 柔和高級系（白、莫蘭迪、裸粉、天藍、暖沙、霧霾綠）。
- **濾鏡：** 自然偏暖輕調色，**保留真實皮膚質感——絕不重磨皮、絕不變形身材**。
- **構圖：** 生活感 / 網感優先；自拍、POV、抓拍 > 棚拍。
- **音樂：** 柔和夢幻、city-pop / 輕電子，深夜文配抒情慢歌。

四個常用真實參考帳號各代表一條細分路線——融合或側重，定義角色個性：

| 參考帳號 | 代表特質 | 融合 / 轉化方向 |
|----------|----------|-----------------|
| @sophieraiin | 網感、純欲風、年輕活潑能量 | 純欲風主軸：清純臉 × 線條身材的反差 |

**設計新角色時：** 在 `meta.reference_profiles` 記下借鑒帳號，並在 `character.md` 用一張
「參考帳號 → 借鑒元素 → 本角色轉化方式」的表格說明設計關係（見 `kols/chloe-lin/character.md`）。

## 7. 品牌合作邊界 (Brand Do / Don't)

**會合作：** 乾淨現代美感的時尚 / 泳裝 / 運動服飾、clean-girl 美妝保養、健身 / 機能、
咖啡 / 生活 / 居家 / 旅遊、極簡珠寶配件。

**會拒絕（務必寫進 brand_dont）：**
- 任何露骨 / 成人 / NSFW 內容
- 減肥產品 / 節食文化
- 低俗或物化女性的拍攝框架
- 跟柔和高級色調衝突的廉價快時尚
- 違背角色自然語氣的逐字稿
