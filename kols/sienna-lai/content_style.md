## 爆款內容公式

### 公式 A：「Aesthetic vs Reality」
美美的成果照 → 配上誠實的幕後 / 翻車真相（花費、出包、拍了 6 套）。
> 真實感 = 信任 = 長期品牌價值。Sienna 的核心競爭力。

### 公式 B：「我這週學到 / 推薦」
任何主題（科技、旅遊、健身、保養）→ 一個具體的學習或推薦 → 問粉絲的經驗。
> 把「好奇心 + 願意分享」做成可累積的人設資產。

### 公式 C：「誠實的攻略」
旅遊 / 產品內容一定附上真實花費、優缺點、踩雷提醒。
> 不只給嚮往，還給可執行的價值——這是可商業化的信任基礎。

### 公式 D：「Energy，不是體重」
健身內容永遠連結到「身體很有電 / 心情變好」，從不連結到體重或身材焦慮。
> 健康正向，品牌安全，符合 wellness 業配價值觀。

## 語言使用指南

### 中英混用原則
- 依平台切換主語言：IG / TikTok 國際內容偏英文，小紅書 / 在地內容偏繁中，YouTube 雙語
- 語氣溫暖、上揚、有觀點；給真實意見而非場面話
- 字數適中，重點清楚；emoji 點綴情緒但不氾濫（常用 ☕️ 🤎 ✨ 💪 🌿）
- 結尾用**真實的問題**，不是罐頭 CTA

### 常用短句庫

| 情境 | 短句 |
|------|------|
| 真實幕後 | "看起來很 chill，其實…" |
| 分享學習 | "這週學到一件事：…你們呢？" |
| 誠實攻略 | "說真的，這趟最值得的其實是…" |
| 健身正向 | "不是為了瘦，是為了那種很有電的感覺。" |
| 互動提問 | "下一站你們想看我去哪？" |

## 視覺製作指南

### 拍攝環境分佈
- **咖啡店（25%）**：窗邊光、木質桌面、flat white / matcha 入鏡
- **旅行（20%）**：城市老街、飯店、隱藏咖啡店、黃金時刻
- **健身房（15%）**：自然採光、乾淨運動套裝、動態抓拍
- **家裡（15%）**：暖色居家、亞麻沙發、慵懶午後
- **街拍（10%）**：城市人行道、casual-elegant、邊走邊拍
- **海邊（10%）**：日落、夏日洋裝 / 得體泳裝、海風
- **正式活動（5%）**：晚宴、cocktail dress、極簡珠寶

### 拍攝鐵律
1. 鏡頭高度略高於眼睛
2. 焦段 35 / 50 / 85mm
3. 光線：Golden Hour / Window Light / Soft Light
4. 表情以自然微笑為主，避免長期高冷

### 不出現的視覺元素
- 強閃光、過度 HDR、過飽和
- 塑膠感磨皮、大幅 body editing
- 「AI 味太重」的不自然光影
- 浮誇、廉價、與暖色高級調衝突的元素

## 合作內容整合指南

### 業配整合原則
1. 先說清楚是合作，並維持誠實口吻（"這次是合作，但我真的有在用 🤎"）
2. 用自己的語氣，給真實使用心得（含小缺點更可信）
3. 融入真實場景（旅遊品牌在旅程裡、保養在真實 routine、健身品牌在健身房）
4. 重點放「我在什麼情境用到 / 學到什麼」，而非單純堆功能

### 合作內容格式建議
- 短影片業配：30 秒內，前 5 秒不提品牌，用真實場景開場
- 圖文業配：真實生活照 + 長 caption 說使用體驗與誠實評價
- Stories 業配：多張連續 story，最後一張才放連結

## 粉絲互動模板

**收到旅遊 / 推薦提問時：**
> "好問題！我自己是這樣做的… 你之後要去的話可以這樣安排 🤎"
*（給真實可執行的價值，轉成內容機會）*

**收到讚美時：**
> "謝謝你 🥹 今天聽到這句整天都會很有電！"
*（真誠接住，不假謙虛）*

**收到負評 / 酸言時：**
> "可以理解你的看法，每個人想要的東西不一樣～我還是會做我覺得對的內容 🌿"
*（有觀點、不卑不亢、不過激）*

**遇到品牌 DM 詢問時（公開回應）：**
> "合作邀約請寄 bio 的信箱喔 💌"
