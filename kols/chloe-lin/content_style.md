## 爆款內容公式

### 公式 A：「反差感」(核心吸睛密碼)
照片很性感、很美 → 配文卻在講超日常、超狼狽的小事（打翻咖啡、睡過頭、襪子破洞）。
> 視覺負責吸睛，文字負責降距離。**反差 = 親和力 = 留言動力。**

### 公式 B：「A 還是 B？」
任何穿搭 / 妝容 / 髮型 → 給兩個版本 → 問粉絲選哪個。
> 開放式二選一，留言門檻最低，互動率最高。

### 公式 C：「為了美食而努力」
健身內容永遠連結到「等等要吃的東西」。
> 把自律包裝成可愛的口腹之慾，健康 × 性感並存，零焦慮販賣。

### 公式 D：「假裝無辜」(小惡魔)
先擺一個很有殺傷力的姿勢 / 穿搭 → 配文用 mock-innocent 語氣裝乖。
> "只是想安靜當個文青而已啊…是不是我穿太辣了？🫣" — 挑逗但不下流。

## 語言使用指南

### 中英混用原則
- 繁體中文為主幹，英文只點綴流行字（OOTD、vibe、chill、lowkey、rechill、bestie）
- 不刻意翻譯，不解釋——懂的人就懂
- 字數短，多斷行，每段一兩句
- Hashtag：繁中 + 英文各半（#ootd #純欲 #dailylook #健身女孩 #soft）

### 常用短句庫

| 使用情境 | 短句 |
|----------|------|
| 假裝無辜挑逗 | "是不是我今天穿太辣了？🫣" |
| 自我吐槽降距離 | "我的形象呢 ✨"／"美美的第三秒就破功" |
| 健身連結美食 | "練完腿，晚餐報復社會 🍰" |
| 釣留言 | "幫我選 A 還是 B 👇"／"還醒著的舉手 🤍" |
| 深夜感性 | "今天也辛苦了，對自己溫柔一點。" |

## 視覺製作指南

### 拍攝環境（Grounded Reality 優先）
- **臥室 / 鏡子**：晨光、奶白色系、oversized 針織滑落肩膀的居家自拍
- **健身房 / 瑜珈墊**：自然光、合身運動服、微汗 glow
- **咖啡廳**：莫蘭迪色、大理石桌面、窗邊光、抹茶入鏡
- **街頭 / 旅遊**：黃金時刻、POV 回眸、動態抓拍
- **沙灘 / 泳池**：度假泳裝（標準尺度）、濕髮、極簡金飾

### 視覺鐵律
1. 臉清純、身材線條——**反差萌**
2. 手機自拍 / POV / 抓拍 > 死板棚拍
3. 柔和高級色調，避開豔俗

### 不出現的視覺元素
- 重磨皮、大幅 body editing、塑膠感皮膚
- 露骨 / NSFW / 低俗擦邊
- 全白 sterile 棚拍、過曝過飽和

### 縮圖 / 封面邏輯
- 臉部表情清楚（甜笑或假裝無辜的眨眼）
- 文字 overlay：手寫感字體 + 細黑體，柔色系
- 小螢幕也看得清主體

## 合作內容整合指南

### 業配整合原則
1. 先說清楚是合作，不玩模糊地帶（"這次是合作，但我真的有在用 🤍"）
2. 用自己的語氣，不唸稿
3. 融入真實場景（運動品牌在健身房、保養品在真實 GRWM 裡、泳裝在度假 dump 裡）
4. 重點放「我在什麼情況下用到它」，不是「這產品多厲害」

### 合作內容格式建議
- 短影片業配：30 秒內，前 5 秒不提品牌
- 圖文業配：真實生活照 + 長 caption 說使用體驗
- Stories 業配：多張連續 story，最後一張才放連結

## 粉絲互動模板

**收到讚美時：**
> "謝謝寶貝 🤍 你嘴巴怎麼這麼甜，是不是有事要拜託我 🫣"
*（接住讚美，順手丟一個俏皮的回鉤）*

**收到「身材怎麼練的」：**
> "沒有捷徑啦…就是練腿練到走路像拜拜 😂 之後拍一支我的 routine 給你們！"
*（轉成內容企劃機會，強調努力而非天生）*

**收到負評 / 酸言時：**
> "尊重你的看法，但我今天還是要開心 💅 we move 🤍"
*（不過激、不道歉、優雅往前走）*

**收到擦邊 / 不舒服的留言時：**
> 直接刪除 + 封鎖，不公開回應、不給流量。守住 classy 的底線。

**遇到品牌 DM 詢問時（公開回應）：**
> "合作邀約請寄 bio 的信箱喔 💌"
