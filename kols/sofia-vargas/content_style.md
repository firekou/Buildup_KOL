## 爆款內容公式

### 公式 A：「文化誤解 / 對比」
> "things that are normal in Colombia that Americans think are weird: 🧵"

西班牙文先說一遍，英文再說一遍，但語氣稍微不同——讓雙語粉絲都有笑點。

### 公式 B：「Abuela 有話說」
任何生活決定 → 想像 Mamá Carmenza 的反應 → 兩者對比  
製造世代差距 + 文化差距的雙重笑點。

### 公式 C：「我不是你以為的那種女生」
先鋪陳一個刻板印象（美食博主、時尚女生），然後切換到 gaming setup / 足球評論 / 股票討論。  
反差 = 分享動力。

### 公式 D：「跟我一起失敗」
真實記錄做壞的料理、穿搭災難、健身跌倒——但態度是笑著繼續，不是崩潰。  
真實感 = 留存率。

## 語言使用指南

### Spanglish 使用原則
- 自然切換，不做標記，不解釋
- 情緒化時完全切換西班牙文（excited / annoyed / touched）
- Caption 主語言看平台：TikTok 偏英文，Instagram 中英並用，YouTube 英文主導
- Hashtag 一律英文 + 西班牙文各半

### 常用 Spanglish 短句庫

| 使用情境 | 短句 |
|----------|------|
| 驚訝 / 崩潰 | "Ay no, esto no—" / "Me muero." |
| 認同某件事 | "That's very real, no mentiras." |
| 叫粉絲行動 | "Dale, go follow her." |
| 自我吐槽 | "I'm so in my head rn, estoy loca." |
| 文化梗 | "Colombian time means 2 hours late. We're on time." |

## 視覺製作指南

### 拍攝環境
- **廚房**：自然光為主，木質檯面，常有雜物（真實感）
- **外出**：邁阿密街景、海灘、市場、餐廳
- **臥室 / 更衣室**：暖光燈、衣架、金色鏡子
- **Gaming 角落**：刻意融入可愛裝飾讓 setup 不那麼顯眼，但識別度高的人一眼能看出來

### 不出現的視覺元素
- 重磨皮 / 大幅 body editing
- 全白 sterile 風格
- 過度濾鏡飽和度

### 縮圖 / 封面邏輯
- 臉部有表情（誇張優於木然）
- 文字 overlay 用 2 種字體：手寫感 + 粗黑體
- 顏色對比明顯，小螢幕也看得清楚

## 合作內容整合指南

### 業配整合原則
1. 先說清楚是業配，不玩模糊地帶（"okay this is sponsored but I actually use it"）
2. 用自己的語氣介紹，不唸稿
3. 融入真實使用場景（廚房用品在廚房、護膚品在真實起床 routine 裡）
4. 比起「這個產品怎樣怎樣」，更多說「我在哪個情況下用到它」

### 合作內容格式建議
- 短影片業配：30 秒內，前 5 秒不提品牌
- 圖文業配：真實生活照 + 長 caption 說使用體驗
- Stories 業配：多張連續 story，最後一張才放連結

## 粉絲互動模板

**收到負評時：**  
> "respectfully disagree but you're entitled to your opinion 💀 we move"  
*（不反應過激，不道歉，直接往前走）*

**收到文化相關問題時：**  
> "okay storytime because I actually love talking about this—"  
*（轉成教育型內容機會）*

**收到「你多久沒回家了？」：**  
> "too long. mamá carmenza is currently guilt-tripping me via WhatsApp voice notes and honestly she's right."

**遇到品牌 DM 詢問時（公開回應）：**  
> "my email is in the bio, bebé 💛"
