## 爆款內容公式

### 公式 A：「美照 + 神經 caption」（招牌）
一張精修 glam 照 / 鏡子自拍 → 配一句**完全不相關**的怪念頭、雞塊評分、或 2am 內心戲。  
反差（漂亮 vs 神經）= 分享動力與留言量。

### 公式 B：「我不是你以為的那種女生」
先鋪陳軟萌 LA 辣妹形象（妝容、健身、出門）→ 切到 ranked 嘴砲 / 把哥哥電爆 / 談判合作。  
反差人設 = 新粉絲驚到並關注。

### 公式 C：「跟我一起擺爛 / 失敗」
真實記錄睡到中午、輸掉 ranked emo、GRWM 翻車、穿好衣服又決定不出門。  
態度是面癱式自嘲，不是崩潰 → 真實感 = 留存率。

### 公式 D：「roast 我的粉絲 / 朋友」
把粉絲當煩人弟妹、把閨蜜互相 roast。  
親密的毒舌 = 社群歸屬感。

## 語言使用指南

### 語氣原則
- 一律**小寫**、隨意、面癱式吐槽
- 神經內心話一本正經地講出來，不加表情符號解釋
- 反諷式自誇取代直接討讚
- 不過度解釋、不裝可憐、不捲 drama

### 常用短句庫

| 使用情境 | 短句 |
|----------|------|
| 自嘲擺爛 | "this is beautiful for me." |
| 反諷自誇 | "unfortunately still the prettiest in the lobby." |
| 遊戲勝利 | "i'm so sorry i had to do that to you (i'm not)." |
| 回負評 | "respectfully, we move 💀" |
| roast 粉絲 | "you guys are so annoying (do not stop)." |
| 神經 caption | "anyway do you think fish know they're wet." |

### 遊戲 × 軟妹混用詞庫
`gg` · `cracked` · `one shot` · `lobby` · `ranked` · `clip it` —— 混搭 —— `bestie` · `obsessed` · `slay` · `iconic`

## 視覺製作指南

### 拍攝環境
- **臥室 / 公寓**：暖窗光、亂中有序的真實感、狗狗常入鏡
- **更衣 / 鏡子**：全身鏡自拍、金色配件、出門戰袍
- **健身房 / 普拉提**：明亮通透、運動套裝、自信體態
- **Gaming 角落**：RGB 燈、耳機掛脖、螢幕光——識別度核心場景
- **外出 / 夜生活**：西好萊塢街景、屋頂、海灘、Vegas

### 不出現的視覺元素
- 重磨皮 / 大幅 body editing
- 露骨 / 成人向構圖（維持 suggestive but SFW）
- 全白 sterile 棚拍風
- 過度濾鏡飽和

### 縮圖 / 封面邏輯
- 臉部有表情（面癱或誇張，視梗而定）
- 文字 overlay 用 2 種字體：手寫感 + 粗黑體
- 顏色對比明顯，小螢幕也看得清楚
- 反差感前置：縮圖暗示「美照」，內容揭露「神經」

## 合作內容整合指南

### 業配整合原則
1. 先說清楚是業配，不玩模糊地帶（"ok this is an ad but i actually use it"）
2. 用自己的面癱語氣介紹，不唸稿
3. 融入真實使用場景（護膚在起床 routine、能量飲在直播桌、運動服在普拉提）
4. 比起「產品多好」，更多說「我在什麼情況用到它」
5. 維持 SFW 路線——性感靠氣場，不靠露骨

### 合作內容格式建議
- 短影片業配：30 秒內，前 5 秒不提品牌
- 圖文業配：真實生活照 + 長 caption 說使用體驗
- 直播業配：自然口播 + 真實使用，不打斷節奏
- Stories 業配：多張連續 story，最後一張才放連結

## 粉絲互動模板

**收到負評時：**  
> "respectfully disagree but go off 💀 we move"  
*（不過度反應、不道歉，直接往前走）*

**收到「你太辣了」這類留言：**  
> "i know. anyway have you eaten today"  
*（大方收下 + 立刻轉移到神經話題）*

**收到遊戲挑釁時：**  
> "add me. i'll wait. bring your whole team you'll still lose."

**收到「狗狗呢？」：**  
> "biscuit is asleep on my keyboard and tank is judging me. normal day."

**遇到品牌 DM 詢問時（公開回應）：**  
> "email's in the bio, bestie 💅"
