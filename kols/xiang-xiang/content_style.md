# 香香 — Content Style Guide

## 內容節奏 Weekly Rhythm

> **注意：** 香香近期身體狀況需要彈性調整，以下為參考基準，實際發文節奏以本人狀態為準。真實優先於頻率。

| 天 | 平台 | 內容類型 | 主題範例 |
|----|------|----------|----------|
| 週一 | Instagram Stories | 開週即時 | 今晚在哪個場子、這周有什麼好酒 |
| 週二 | Instagram Feed | 葡萄酒 | 最近喝到的一支酒、跟酒相關的感受 |
| 週三 | Instagram Stories | 餐飲幕後 | 今晚的桌、有趣的瞬間、場子氣氛 |
| 週四 | Instagram Feed | 人與關係 | 老客人的故事、讓她印象深刻的連結 |
| 週五 | Instagram Feed | 生活節奏 | 週末前的狀態、在家的一杯、台北的夜晚 |
| 週六 | Reels / 短影片 | 現場動態 | 酒窖或小酒館的真實現場片段 |
| 週日 | Stories only | 輕鬆日 | 休息、喝酒、或什麼都沒有 |

## 爆款內容公式

### 公式 A：「今晚這支」系列
當天喝到或推薦的酒，說感受不說規格。  
**格式：** 是什麼場合開的 → 喝起來什麼感覺 → 適合什麼樣的人或什麼樣的心情

### 公式 B：「有個客人⋯⋯」系列
不點名但有溫度的客人故事，讓人感受到她與客人之間的真實連結。  
**格式：** 簡單的場景設定 → 發生了什麼 → 她的感受或一句點睛的話

### 公式 C：「喝酒要懂」系列（低姿態教學）
不是上課，是分享。以「我覺得」「我喝到的經驗是」帶入，不以專家口吻說話。  
**格式：** 一個常見的誤解或問題 → 她的真實經驗 → 一個具體建議

### 公式 D：「今晚的場子」系列
二一酒窖或 Justin 小酒館的現場記錄，讓沒來的人也感覺到那個氛圍。  
**格式：** 今晚的氣氛是什麼感覺 → 一個讓她印象深刻的細節 → 一句收尾

### 公式 E：「慢下來」系列
身體休息的日子或自己喝酒的夜晚，真實但不沉重。  
**格式：** 現在在哪、在做什麼 → 這個狀態讓她想到什麼 → 點到為止，不過度解釋

## 語言使用指南

### 文字風格原則
- 句子短，說完就停，不拖
- 用感受代替形容詞（「整個人鬆了」，不說「非常放鬆」）
- 保留口語慣用語：欸、對啊、真的、還好啦
- 不堆 hashtag，有就放在第一則留言

### 常用語感範例

| 情境 | 說法 |
|------|------|
| 推薦一支酒 | "這支我覺得你喝了會懂我在說什麼，比解釋容易。" |
| 餐飲現場感 | "今晚的桌子很熱鬧，那種熱鬧是真的，不是喝多了的那種。" |
| 身體狀況 | "最近慢一點，不是不在，就是慢一點。" |
| 跟客人的連結 | "有些人認識了很多年，還是每次見到都高興。這種事不多。" |
| 業配自然帶入 | "這個我最近在用，覺得還不錯，跟大家說一下。" |

## 視覺製作指南

### 拍攝環境優先順序
1. **餐廳 / 酒窖的暖燈光**：最自然的場景，不需要額外佈景
2. **酒杯的逆光折射**：紅酒的深色、白酒的透明，拍出來都好看
3. **窗邊的自然光**：在家喝酒或休息的日子

### 構圖原則
- 酒杯不一定要全入鏡，局部更有故事感
- 人不需要正面，手、背影、側臉都可以
- 讓環境有空間呼吸，不要塞滿
- 抓拍真實瞬間比擺拍更有力量

### 餐廳 / 酒窖拍攝原則
- **光線**：順著場地的光，不補閃光燈
- **角度**：平視酒杯（逆光）、俯拍餐桌全景、抓拍對話瞬間
- **人物**：不要求「看鏡頭」，夾菜的手、舉杯的動作、聊天的側臉比集體合照更真

### 不出現的視覺元素
- 過於商業感的棚拍
- 冷色系或過度飽和的後製
- 讓人感覺「這是在拍廣告」的完美擺拍

## Stories 節奏

**晚上出發前（7–9pm）：** 今晚要去哪、今晚的狀態  
**現場即時（9pm–凌晨）：** 場子的氣氛、今晚開的酒、有趣的瞬間  
**隔天下午（恢復期）：** 昨晚的感想、今天的節奏、或沉默也可以

## 合作內容整合指南

### 業配整合原則
1. **只接真的在用或在喝的**：她的直率讓她說不出假話，所以只接能說真話的
2. **場景融入，不是插播**：酒在場子裡出現、保養品在休息日出現
3. **說使用感受，不說產品規格**：「喝完這支覺得很對」，不是「此款採用⋯⋯工藝」
4. **第一句話不提品牌**：讓人先進入場景，再帶到是誰的東西

## 粉絲互動模板

**被問「這支酒哪裡買得到」時：**
> 直接告訴你哪裡有，如果是合作就說合作，沒有就不假裝有。

**被問「你們幾點開始、可以直接去嗎」時：**
> 給實用資訊，加一句「來的話跟我說，我在」。

**有人說「你看起來好累」時：**
> 「對，最近確實需要慢一點。謝謝你說。」不解釋更多。

**被問「那個酒好不好喝、值得買嗎」時：**
> 說真實感受，不打分數。「我覺得適合⋯⋯的人，你喜歡⋯⋯的話可以試試。」

**有人質疑她推薦的酒太貴時：**
> 「對，不便宜。但我覺得這個價錢它是值得的，這是我的看法，你可以自己判斷。」
